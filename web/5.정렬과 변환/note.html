CSS3 스타일로 태그에 동적 변화 만들기

자바스크립트로 구현하던 것을 CSS3로 작성 가능

3가지 기법을 지원한다.
  1. 애니메이션(animation)
  2. 전환(transition)
  3. 변환(transform)

1. 애니메이션

HTML 태그의 모양 변화를 시간 단위로 설정
1. @keyframes 로 HTML 태그의 시간별 모양 변화 만들기

2. 애니메이션 스타일 시트 작성


    @keyframes textColorAnimation {
        0% { color: blue; }
        30% { color: green; }
        100% { color: red; }
    }

    span {
        animation-name: textColorAnimation;
        animation-duration: 3s;
        animation-iteration-count: infinite;

        font-size: 50px;
        font-weight: bold;
    }

span태그의 텍스트를
blue, green, red로 무한 반복 


    @keyframes bomb {
        from { font-size: 500%; }
        to { font-size: 100%; }
    }
    
    span {
        animation-name: bomb;
        animation-duration: 3s;
        animation-iteration-count: infinite;

        font-weight: bold;
        background-color: yellow;
    }

3초동안 감소하는 텍스트(반복)




2. 전환

HTML 태그에 적용된 CSS3 프로퍼티 값의 변화를 서서히 진행시켜 애니메이션 효과 생성
HTML 태그의 색이나 모양, 위치 등이 서서히 변화는 효과
전환 설정: transition 프로퍼티 이용

커서를 올리면 5초에 걸쳐 확대



3. 변환

텍스트나 이미지를 회전, 확대 등
다양한 기하학적인 모양으로 출력할 수 있게 해준다
회전 각도의 단위는 deg이며, 시계방향으로 회전한다

transform에 사용 가능한 2차원 변환 함수들

마우스를 누르면 변환하게 해두었다
