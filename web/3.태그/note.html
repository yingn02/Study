HTML 기본 문법

<p>Hello</p>
위와 같은 것을 '요소 (element)'라고 하며,
요소는 시작태그, 콘텐츠, 종료태그로 이루어져 있다.
요소는 중첩이 가능하다.
br, hr, img 와 같이 content가 없는 요소를 '빈 요소'라고 한다.

<img src="html.png">
속성은 요소의 성질이며, 특징을 정의한다.
이름 = 값 의 형태로 이루어져 있다.
src는 어트리뷰트명, html.png는 어트리뷰트값에 해당한다.

<!--주석-->
위와 같은 방법으로 주석을 달 수 있다.

Semantic Web

Not semantic tag

It's semantic tag

위의 두 문장은 같아보이지만 다른 태그를 썼다.
위 문장은 아무 의미없이 글씨크기를 5로 설정하고 굵게 표시한 것뿐이지만,
아래 문장은 h2 태그로 감싼 의미있는 문장이다. Semantic tag를 사용한 것이다.
즉, 시맨틱 태그란 '브라우저, 검색엔진, 개발자 모두에게 콘텐츠의 의미를 명확히 설명'하는 태그다.
검색엔진 로봇은 크롤링과 인덱싱을 이용하여 검색결과를 보여주기 때문에 의미있게 태그를 사용하는 것이 중요하다.
semantic element: div, span, ...
non-semantic element: form, table, img, header, nav, footer, ...
