  1. HTML 태그의 이벤트 리스너 속성에 작성
  2. script 태그에 작성
  3. 자바스크립트 파일에 작성
  4. URL 부분에 작성

자바스크립트 식별자

  • 첫 번째 문자: 알파벳, 언더스코어, $ 문자만 가능
  • 두 번째 문자: 위를 포함하여, 숫자도 가능
  • 대소문자는 구분되어 다루어짐
  • 자바스크립트 예약어 사용 불가

자바스크립트 문장

  • 문장과 문장을 구분하기 위해 세미콜론 사용
  • 주석문은 // 와 /* */ 사용 가능

데이터 타입

자바스크립트 언어에서 다루는 데이터 종류

  • 숫자 타입: 정수, 실수
  • 논리 타입: 참, 거짓
  • 문자열 타입: "문자열"
  • 객체 레퍼런스 타입: 객체를 가리킴, C언어의 포인터와 유사
  • null: 값이 없음을 표시하는 특수 키워드

특징

  • 자바스크립트에는 문자('a') 타입이 없다. 문자열("a")로 표현한다.

변수, 상수

변수 선언: var, let
상수 선언: const
자바스크립트에는 변수 타입 없음

변수의 사용 범위와 생명

선언사용 범위변수의 생명
전역 변수함수 밖에서 선언 혹은 var/let 키워드 없이 아무 곳에서나 선언프로그램 전역프로그램이 실행을 시작할 때 생성, 프로그램 종료할 때 소멸
지역 변수함수내에 let으로 선언선언된 함수 내함수가 실행될 때 생성, 함수가 종료될 때 소멸
블록 변수let으로 if, while, for 등 블록 내에 선언선언된 블록 내블록의 실행 시작 시 생성, 블록이 끝나면 소멸

this로 전역변수 접근

지역 변수와 전역 변수의 이름이 같은데
전역 변수에 접근하고자 할 때 : this.전역변수

단, let으로 선언된 전역 변수는 this로 접근 불가

let의 특징

var x = 1;
var x = 2; // 새로운 변수 x 생성, 개발자의 실수로 코딩 오류 발생

let x = 1;
let x = 2; // 오류, 변수 x에 대한 재선언 불가능

let은 변수 사용 범위를 블록 내로 제한

상수

const MAX = 10; // 10의 값을 가지는 상수 MAX 선언

선언된 후 값 수정 불가능

상수 재선언 불가능

블록 범위에서만 사용

자바스크립트의 연산자

산술 연산자

더하기(+), 빼기(-), 곱하기(*), 나누기(/), 나머지(%)
연산의 결과는 항상 실수다.

증감 연산자

변수 a가 있다면,
a++ : a 반환 후 a 1증가
a-- : a 반환 후 a 1감소
++a : a 1증가 후 a 반환
--a : a 1감소 후 a 반환

대입 연산자

대입 연산: 오른쪽 식의 결과를 왼쪽 변수에 대입
대입 연산자 종류

비교 연산자

두 값을 비교하여, true 나 false의 결과를 내는 연산
비교 연산자 종류

논리 연산자

AND: && / 두 값이 모두 true 일 때 true 리턴
OR: || / 둘 중 하나라도 true 면 true 리턴
NOT: !a / a가 true면 false 값을, false이면 true 값 리턴

조건 연산자

let big = (x>y) ? x : y;
// (x>y) 가 true면 x 반환, false면 y 반환

비트 연산

비트 논리 연산
AND: & / 두 비트 모두 1이면 1, 아니면 0
OR: | / 두 비트 모두 0이면 0, 아니면 1
NOT: ~a / a가 1이면 0으로, 0이면 1로
XOR: ^ / 두 비트가 다르면 1, 같으면 0

비트 시프트 연산
시프트: 저장 공간에서 비트들의 오른쪽/왼쪽 이동

문자열 연산자

"abc" + "de" = "abcde"
"abc" + 23 = "abc23"
23 + "abc" = "23abc"

순서에 유의하도록 한다
23 + 35 + "abc" = "58abc"
"abc" + 23 + 35 = "abc2335"

문자열에서도 비교 연산자를 사용할 수 있다

자바스크립트의 비교문

if, if-else

예시
        let grade;
        let score = prompt(
        ＂한성대님 점수를 입력하세요", 100);
        score = parseInt(score); // 문자열을 숫자로 바꿈
        if(score >= 90) // score가 90 이상
            grade = "A";
        else if(score >= 80) // 80 이상 90 미만
            grade = "B";
        else if(score >= 70) // 70 이상 80 미만
            grade = "C";
        else if(score >= 60) // 60 이상 70 미만
            grade = "D";
        else // 60 미만
            grade = "F";
        document.write(score + "는" + grade + "입니다.")

switch

예시
        let fruits="사과";
        switch(fruits) {
            case "바나나": 
                price = 200; break;
            case "사과":
                price = 300; break;
            case "체리":
                price = 400; break;
            default:
                document.write("팔지 않습니다.");
                price = 0;
        }
        // switch 문의 실행 결과 price=300

case문의 '값'은 const로 선언된 상수나 리터럴만 가능
case문의 '값'에 변수나 식은 사용 불가

자바스크립트의 반복문

for 문

예시
        for(let i = 0; i < 0; i++) {
            document.write(i);
        }

while 문

예시
        let i = 0;
        while(i < 10) {
            document.write(1);
            i++;
        }

do-while 문

예시
        let i = 0;
        do {
            document.write(1);
            i++;
        } while(i < 10);

반복문 내의 break 문과 continue 문

break 문: 가장 안쪽 반복문 하나만 벗어나도록 함
continue 문: 반복 코드 실행을 중단하고, 다음 반복으로 건너뜀

자바스크립트에서 제공하는 전역 함수

전역 함수명설명

isFinite(value)value 가 숫자면 true 리턴
isNON(value)value 가 숫자가 아니면 true 리턴
